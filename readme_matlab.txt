如果电脑上没有安装MATLAB：# 自动安装 MATLAB RUNTIME

	1. 首先打开 stressbri_matlab 文件夹

	2. 双击 MyAppInstaller_web.exe 文件，等待图形化界面 version_*_* Installer 出现

	3. 浏览该版本功能变化，点击“下一步”

	4. 选择软件安装目录，并添加桌面快捷方式（推荐）

	5. 确认安装

如果电脑上有装MATLAB：# 手动打开自己的MATLAB
		
	1. 找到m文件夹，将其中所有文件复制到工作文件夹中

	2. 直接在MATLAB的脚本编辑器里打开并运行
